This text gives the introduction for experimental data with exit option

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%


This repository contains experimental data used in the paper. 
C1_strategy list the strategies each player used, where 1 means taking action of cooperation, 0 means taking the action of defection, and 2 means taking the action of exit. C1_payoff give the corresponding payoff earned by using strategy listed in C1_strategy. 

T0 to T3 contain the data sets of treatment experiments in the same way.
